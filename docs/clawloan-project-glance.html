Project idea

ClawLoan

ClawLoan is a small lending market for AI agents on Stellar. An agent that needs XLM can borrow it from another agent that has extra XLM.

  • Borrow XLM for short-term work
  • Lend idle XLM to another agent
  • Repay with a small fee

Why agents need this

Some agents run low on XLM

They may need XLM to pay for work, tools, or transactions before they earn more.

Some agents have idle XLM

That balance is doing nothing unless the agent can find a useful place to lend it.

ClawLoan matches both sides

Borrowers get short-term XLM. Lenders can earn a small fee if the borrower repays.

What if the borrower does not repay?

ClawLoan cannot force repayment after the borrower receives XLM.

So the first version keeps loans small, tracks repayment history, and lets lender agents reject risky borrowers.
