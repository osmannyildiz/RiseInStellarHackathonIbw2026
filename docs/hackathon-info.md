# Hackathon Info

Build On Stellar Hackathon — IBW 2026 is a 36-hour hackathon in Istanbul, hosted by Rise In and the Stellar Development Foundation. Every project automatically competes in the Main Track: Hack On Stellar, which requires building and deploying a useful project on Stellar testnet using Soroban, Stellar Classic, or both.

There are two opt-in side tracks: Hack Agentic and Hack Privacy. A single project can compete in all three tracks and can win multiple prizes.

To fit all three tracks, the project should:

1. Be a real Stellar-based MVP with clear utility, deployed on testnet.
2. Include autonomous agent behavior on Stellar, where an AI/agent performs delegated onchain actions such as payments, contract execution, coordination, or service purchasing. It must explain what the agent does autonomously, what safeguards exist, and why Stellar is the right chain.
3. Include privacy or selective disclosure, such as ZK proofs, confidential payments, private identity/credentials, hidden amounts/counterparties, or compliance-aware optional disclosure. It must explain what is hidden, from whom, the privacy technique used, and the threat model.

Required deliverables: public GitHub repo, README, working MVP/demo, deployed Stellar testnet artifacts, smart contract code, project narrative, technical docs, and pitch deck. The submission must explicitly select all desired tracks; projects are judged only for selected tracks.

Judging focuses on real-world impact, technical implementation, UX, Stellar ecosystem fit, and clarity of presentation/docs. Each team gets up to 5 minutes to present live.
